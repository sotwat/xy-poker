# Antigravity Rules - XY Poker Customizations

## 1. アプデ即時反映の検証ルール
- デプロイ時には必ず、ロビー画面のバージョン番号が更新されていることをブラウザで検証すること。
- キャッシュをパージするか、アセットURLにハッシュクエリを付与して古いアセットの読み込みを防ぐこと。

## 2. レイアウト設計ガイド
- 本ゲームは幅 480px, 高さ 850px の .app コンテナ内に制限されています。
- レイアウトに viewport 単位 (vh/vw) は使用禁止です。代わりに container units (cqh/cqw) や % を使用してください。
